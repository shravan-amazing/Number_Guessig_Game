Number Guessing Game (Python)

A simple command-line game where the user tries to guess a randomly generated number between 1 and 100.

How to Run

Install Python 3.

Run the program:

python number_guessing_game.py

How It Works

The program generates a random number between 1 and 100.

The user inputs guesses until they find the correct number.

Feedback is given if the guess is too low or too high.

The game ends when the correct number is guessed.
